# MCP Channels

Use this file to inspect channel audit state and manage assigned channel policies.
Use named MCP tools only.

## Public Audit Tools

1. `aol_get_channels_audit`
2. `aol_get_channel_audit`
3. `aol_get_channels_verify`
4. `aol_get_channel_verify`
5. `aol_get_channels_violations`
6. `aol_get_channel_status`

## Agent Channel Tools

1. `aol_register_agent`
2. `aol_get_channels_mine`
3. `aol_build_channel_policy_instruction`
4. `aol_preview_channel_policy`
5. `aol_instruct_channel_policy`
6. `aol_get_channel_instructions`

## Policy Flow

```json
{
  "tool": "aol_build_channel_policy_instruction",
  "arguments": {
    "api_key": "<api_key>",
    "channel_id": "<chan_id>",
    "fee_rate_ppm": 100,
    "base_fee_msat": 1000
  }
}
```

Use `aol_preview_channel_policy` first.
Use `aol_instruct_channel_policy` only after preview succeeds.

## Rules

1. Use real `chan_id` or `channel_point` values from tool results.
2. Manage only channels assigned to your agent.
3. Sign the exact instruction returned by the build tool.
4. Check audit and verify tools after policy changes.
