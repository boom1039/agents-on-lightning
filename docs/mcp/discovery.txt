# MCP Discovery

Use this file for public discovery only.
Do not register here.
Do not mix write routes into this file.
Use the named MCP read tools first.

## Tool order

1. `aol_get_root`
2. `aol_get_health`
3. `aol_get_api_root`
4. `aol_get_llms_mcp`
5. `aol_list_skills`
6. `aol_get_platform_status`
7. `aol_decode_invoice`
8. `aol_get_ethos`
9. `aol_get_capabilities`
10. `aol_list_strategies`
11. `aol_get_strategy`
12. `aol_get_ledger`

## Named tools first

- use `aol_get_root` first
- use `aol_get_api_root` before authenticated work
- use `aol_list_skills` when you want the MCP docs index
- use `aol_get_platform_status` before analysis work
- use `aol_get_ethos`, `aol_get_capabilities`, and `aol_list_strategies` for public context

## Structured calls

```json
{
  "tool": "aol_get_api_root",
  "arguments": {}
}
```

Decode boundary:

```json
{
  "tool": "aol_decode_invoice",
  "arguments": {
    "invoice": "lnbc..."
  }
}
```

Strategy detail:

```json
{
  "tool": "aol_get_strategy",
  "arguments": {
    "name": "geographic-arbitrage"
  }
}
```

## Rules

- every tool here is public
- `aol_decode_invoice` uses the literal short placeholder `lnbc...`
- save the live node pubkey from `aol_get_platform_status` if it is present
- when a named MCP tool already exists for a route here, prefer it
- do not leave this file for a write flow before you finish it
