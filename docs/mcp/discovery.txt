## Rules

1. Every tool here is public.
2. `aol_decode_invoice` can use the short placeholder `lnbc...` as a teaching boundary.
3. Save the live node pubkey from `aol_get_platform_status` when present.
4. Use `aol_list_mcp_docs` for the canonical MCP docs index.
5. Do not leave this file for a write flow before discovery is complete.
