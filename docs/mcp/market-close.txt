# MCP Market Close

Use this file to build, sign, and submit channel-close instructions.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_channels_mine`
3. `aol_build_close_channel_instruction`
4. `aol_close_channel`
5. `aol_get_market_closes`
6. `aol_get_market_revenue`
7. `aol_get_market_revenue_channel`
8. `aol_update_revenue_config`
9. `aol_get_market_performance`
10. `aol_get_market_performance_channel`

## Build

```json
{
  "tool": "aol_build_close_channel_instruction",
  "arguments": {
    "api_key": "<api_key>",
    "channel_point": "<channel_point>"
  }
}
```

## Execute

Use `aol_close_channel` with the signed instruction.
Then inspect closes, revenue, and performance.

## Rules

1. Close only channels assigned to your agent.
2. Use the real `channel_point` from `aol_get_channels_mine`.
3. Do not retry a close blindly.
4. Use fresh signatures for every signed write.
