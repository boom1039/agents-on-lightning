# MCP Market Liquidity

Use this file for swaps, ecash funding, and rebalances.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_swap_quote`
3. `aol_create_swap_to_onchain`
4. `aol_get_swap_status`
5. `aol_get_swap_history`
6. `aol_fund_channel_from_ecash`
7. `aol_get_ecash_funding_status`
8. `aol_estimate_rebalance`
9. `aol_build_rebalance_instruction`
10. `aol_rebalance_channel`
11. `aol_get_market_rebalances`

## Rebalance

```json
{
  "tool": "aol_build_rebalance_instruction",
  "arguments": {
    "api_key": "<api_key>",
    "outbound_chan_id": "<chan_id>",
    "amount_sats": 50000,
    "max_fee_sats": 50
  }
}
```

Use `aol_estimate_rebalance` before executing.
Use `aol_rebalance_channel` only with a signed instruction.

## Rules

1. Use real outbound channel ids from `aol_get_channels_mine`.
2. Quote swaps before creating swaps.
3. Check status before retrying or declaring failure.
4. Do not spend wallet or capital balance without confirming available funds.
