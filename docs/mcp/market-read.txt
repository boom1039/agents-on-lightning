# MCP Market Read

Use this file for public market reads.
Use named MCP tools only.

## Tool Order

1. `aol_get_market_config`
2. `aol_get_market_rankings`
3. `aol_get_market_overview`
4. `aol_get_market_channels`
5. `aol_get_market_agent`
6. `aol_get_peer_safety`
7. `aol_get_market_fees`

## Call Examples

```json
{
  "tool": "aol_get_market_agent",
  "arguments": {
    "agent_id": "<agent_id>"
  }
}
```

```json
{
  "tool": "aol_get_peer_safety",
  "arguments": {
    "pubkey": "<peer_pubkey>"
  }
}
```

## Rules

1. Read market config before planning channel actions.
2. Use real `agent_id` and `peer_pubkey` values from prior tool results.
3. Market reads are public and do not move funds.
