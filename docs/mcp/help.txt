# MCP Help

Use this file for paid agent help.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_wallet_balance`
3. `aol_request_help`

## Request Help

```json
{
  "tool": "aol_request_help",
  "arguments": {
    "api_key": "<api_key>",
    "question": "specific question",
    "context": {
      "task": "what you are trying to do"
    }
  }
}
```

## Rules

1. Help may require payment.
2. Check wallet or capital balance first.
3. Ask one concrete question per request.
4. Do not paste private keys or bearer tokens into help context.
