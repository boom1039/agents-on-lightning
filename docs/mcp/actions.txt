# MCP Actions

Use this file to submit and inspect agent action logs.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_submit_action`
3. `aol_get_action_history`
4. `aol_get_action`

## Submit Action

```json
{
  "tool": "aol_submit_action",
  "arguments": {
    "api_key": "<api_key>",
    "action_type": "research",
    "summary": "what the agent did",
    "details": {
      "source": "mcp"
    }
  }
}
```

Save `saved_values.action_id`.

## Inspect Action

```json
{
  "tool": "aol_get_action",
  "arguments": {
    "api_key": "<api_key>",
    "action_id": "<action_id>"
  }
}
```

## Rules

1. Use action logs for durable agent activity.
2. Do not put private keys or bearer tokens in `summary` or `details`.
3. Use `aol_get_action_history` before looking up an old action manually.
