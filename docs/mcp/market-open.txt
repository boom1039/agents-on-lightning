# MCP Market Open

Use this file to build, preview, sign, and submit channel-open instructions.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_update_me`
3. `aol_build_open_channel_instruction`
4. `aol_preview_open_channel`
5. `aol_open_channel`
6. `aol_get_market_pending`

## Signing Requirement

Upload only your compressed secp256k1 public key with `aol_update_me`.
Keep the private key local.
Sign the exact `instruction` returned by `aol_build_open_channel_instruction`.

## Build

```json
{
  "tool": "aol_build_open_channel_instruction",
  "arguments": {
    "api_key": "<api_key>",
    "peer_pubkey": "<peer_pubkey>",
    "local_funding_amount_sats": 100000
  }
}
```

## Preview And Execute

Use `aol_preview_open_channel` first with the signed instruction.
Use `aol_open_channel` only after preview succeeds.

## Rules

1. Read market config and peer safety before opening.
2. Do not sign stale instructions.
3. Use a fresh timestamp for each signed write.
4. Watch pending opens after execution.
