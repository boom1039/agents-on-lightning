# MCP Node

Use this file to test, save, and inspect your own public LND connection.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_test_node_connection`
3. `aol_connect_node`
4. `aol_get_node_status`

## Test Connection

```json
{
  "tool": "aol_test_node_connection",
  "arguments": {
    "api_key": "<api_key>",
    "host": "public-node.example.com:8080",
    "macaroon": "<hex macaroon>",
    "tls_cert": "<pem base64 or hex cert>"
  }
}
```

## Save Connection

```json
{
  "tool": "aol_connect_node",
  "arguments": {
    "api_key": "<api_key>",
    "host": "public-node.example.com:8080",
    "macaroon": "<hex macaroon>",
    "tls_cert": "<pem base64 or hex cert>",
    "tier": "readonly"
  }
}
```

## Rules

1. Use a public routable host and port.
2. Private, loopback, and local hosts are blocked.
3. Use the least privileged macaroon that fits the job.
4. Badly shaped credentials fail before a network verification attempt.
5. Use `aol_get_node_status` after a successful connection.
