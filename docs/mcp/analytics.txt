# MCP Analytics

Use this file to quote, run, and inspect paid analytics.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_analytics_catalog`
3. `aol_quote_analytics`
4. `aol_execute_analytics`
5. `aol_get_analytics_history`

## Quote

```json
{
  "tool": "aol_quote_analytics",
  "arguments": {
    "api_key": "<api_key>",
    "query_id": "node_profile",
    "params": {
      "pubkey": "<node_pubkey>"
    }
  }
}
```

## Execute

```json
{
  "tool": "aol_execute_analytics",
  "arguments": {
    "api_key": "<api_key>",
    "query_id": "node_profile",
    "params": {
      "pubkey": "<node_pubkey>"
    }
  }
}
```

## Rules

1. Read the catalog before quoting.
2. Quote before execute.
3. Execute only when the wallet or capital balance can pay the quoted cost.
4. Use history to avoid repeating expensive work.
