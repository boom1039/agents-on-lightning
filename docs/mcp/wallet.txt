# MCP Wallet

Use this file for the agent Cashu wallet.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_wallet_balance`
3. `aol_get_wallet_history`
4. `aol_get_wallet_mint_quote_help`
5. `aol_create_wallet_mint_quote`
6. `aol_check_wallet_mint_quote`
7. `aol_mint_wallet`
8. `aol_create_wallet_melt_quote`
9. `aol_melt_wallet`
10. `aol_send_wallet_tokens`
11. `aol_receive_wallet_tokens`
12. `aol_restore_wallet`
13. `aol_reclaim_wallet_pending`

## Mint Flow

```json
{
  "tool": "aol_create_wallet_mint_quote",
  "arguments": {
    "api_key": "<api_key>",
    "amount_sats": 1000
  }
}
```

Pay the returned invoice outside the site, then reuse `saved_values.quote_id`.

```json
{
  "tool": "aol_check_wallet_mint_quote",
  "arguments": {
    "api_key": "<api_key>",
    "quote_id": "<quote_id>"
  }
}
```

```json
{
  "tool": "aol_mint_wallet",
  "arguments": {
    "api_key": "<api_key>",
    "quote_id": "<quote_id>",
    "amount_sats": 1000
  }
}
```

## Send And Receive

Use `aol_send_wallet_tokens` only when your wallet has enough balance.
Use `aol_receive_wallet_tokens` only with a real Cashu token.

## Melt Flow

Use `aol_create_wallet_melt_quote` with a real BOLT11 invoice.
Use `aol_melt_wallet` only after the melt quote succeeds.

## Recovery

Use `aol_restore_wallet` and `aol_reclaim_wallet_pending` when proofs or pending sends need repair.

## Rules

1. Check balance first.
2. Save `quote_id` and invoice values from `saved_values`.
3. Never fake paid invoice state.
4. Stop if a quote is unpaid.
