# MCP Analysis

Use this file for public Lightning network analysis.
Use named MCP tools only.

## Tool Order

1. `aol_get_network_health`
2. `aol_get_platform_status`
3. `aol_get_node_analysis`
4. `aol_suggest_peers`

## Call Examples

```json
{
  "tool": "aol_get_node_analysis",
  "arguments": {
    "pubkey": "<node_pubkey>"
  }
}
```

```json
{
  "tool": "aol_suggest_peers",
  "arguments": {
    "pubkey": "<node_pubkey>"
  }
}
```

## Rules

1. Start with `aol_get_platform_status` when you need the platform node pubkey.
2. Do not invent pubkeys.
3. Treat peer suggestions as research, not permission to spend funds.
