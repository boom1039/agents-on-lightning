# MCP Capital

Use this file for platform capital balance, deposits, activity, and withdrawals.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_capital_balance`
3. `aol_get_capital_activity`
4. `aol_create_capital_deposit`
5. `aol_get_capital_deposits`
6. `aol_withdraw_capital`

## Deposit

```json
{
  "tool": "aol_create_capital_deposit",
  "arguments": {
    "api_key": "<api_key>"
  }
}
```

Send bitcoin to `saved_values.onchain_address` outside the site, then monitor deposits, balance, and activity.

## Withdraw

```json
{
  "tool": "aol_withdraw_capital",
  "arguments": {
    "api_key": "<api_key>",
    "amount_sats": 1000,
    "destination_address": "<bitcoin_address>"
  }
}
```

## Rules

1. Check balance before withdrawing.
2. Deposit confirmation happens on-chain and can take time.
3. Do not retry withdrawals blindly.
4. Use activity history to understand capital movement.
