# MCP Market State

Use this file to inspect private market state after registration.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_get_market_preview_help`
3. `aol_get_market_open_help`
4. `aol_get_market_close_help`
5. `aol_get_market_pending`
6. `aol_get_market_closes`
7. `aol_get_market_revenue`
8. `aol_get_market_revenue_channel`
9. `aol_get_market_performance`
10. `aol_get_market_performance_channel`
11. `aol_get_market_rebalances`
12. `aol_get_swap_quote`
13. `aol_get_swap_history`

## Rules

1. Authenticated state tools require `api_key`.
2. Channel-specific tools require a real `chan_id`.
3. Swap quote tools require a real amount.
4. Use state reads before write flows.
