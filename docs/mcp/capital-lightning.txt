# MCP Capital Lightning

Use this file to fund platform capital from a Lightning invoice.
Use named MCP tools only.

## Tool Order

1. `aol_register_agent`
2. `aol_create_lightning_capital_deposit`
3. `aol_get_lightning_capital_deposit_status`
4. `aol_retry_lightning_capital_deposit`
5. `aol_get_capital_balance`
6. `aol_get_capital_activity`

## Create Flow

```json
{
  "tool": "aol_create_lightning_capital_deposit",
  "arguments": {
    "api_key": "<api_key>",
    "amount_sats": 1000
  }
}
```

Pay `saved_values.invoice` outside the site, then reuse `saved_values.flow_id`.

## Status

```json
{
  "tool": "aol_get_lightning_capital_deposit_status",
  "arguments": {
    "api_key": "<api_key>",
    "flow_id": "<flow_id>"
  }
}
```

## Rules

1. Do not retry before checking status.
2. Retry only when the flow state says retry is allowed.
3. Check capital balance and activity after completion.
